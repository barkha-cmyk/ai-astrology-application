### Just Double-Click: `START_HERE.bat`

That's it! The script will:
1. ✅ Check if Node.js is installed

3. ✅ Start the local server

## 🎯 What Happens

1. **Double-click `START_HERE.bat`**
2. **Wait 10 seconds** - Server starts automatically
3. **Browser opens** - Application loads at http://localhost:3000
4. **Done!** - Application is running

## 📍 Server URLs

- **Port 3000**: http://localhost:3000 (Default)
- **Port 8080**: http://localhost:8080 (Alternative)

## ⚠️ Important

- **First Time**: Takes 2-3 minutes (installing dependencies)
- **After That**: Starts in 10-15 seconds
- **Keep Server Running**: Don't close the server window
- **Browser Opens**: Automatically after 10 seconds

## 🛑 To Stop Server

- Close the "AI Astrologer Server" window
- OR press `Ctrl+C` in the server window

## 🔧 Alternative Methods

### Method 2: Auto Start Menu
- Double-click `auto-start.bat`
- Choose port (1 for 3000, 2 for 8080)
- Server starts automatically

### Method 3: Direct Start
- Double-click `start.bat` (port 3000)
- Double-click `start-8080.bat` (port 8080)

## ✅ Success Indicators

You'll know it's working when:
- ✅ Browser opens automatically
- ✅ Application loads without errors
- ✅ No "ERR_CONNECTION_REFUSED" error
- ✅ You see the AI Astrologer homepage
